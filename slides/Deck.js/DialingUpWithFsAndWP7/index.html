Getting Started

  • Install the Windows Phone SDK from http://create.msdn.com/en-US/home/getting_started
  • Download one or more F# WP7 VSIX project template from the Visual Studio Gallery (Silverlight) and (XNA by Joh Deneux)

F# Advantages

  • Clean, crisp, and succinct (yet readable) code
  • F# and XAML are both declarative in nature. Worry more about what you want to do rather than how to do it.
  • All the F# features (great for working with data, async, interoperability, etc., etc.)

Disadvantages

  • C# host project
  • Dependency on external FSharp.Core.dll

Caliburn.Micro

  • Small yet powerful
  • Extremely helpful for MVVM and MVP
  • Convention over configuration
  • Automaticlly wire up View/ViewModels
  • Automatically handle tombstoning
  • Event aggregation, view locator, logging, and much more
